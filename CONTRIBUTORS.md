# Contributors to vframetest

We believe in recognizing the individuals and teams who contribute ideas, design, and code that shape vframetest's evolution. This document honors those contributions.

---

## Architecture & Design Contributors

### AnxietyLab
**Contribution**: Phase 4A TUI Real-Time Dashboard Architecture

**Impact**:
- Designed modular, thread-safe Terminal User Interface system
- Architected component separation (core, state management, rendering, input)
- Demonstrated professional-grade implementation with <1% performance overhead
- Influenced cross-platform compatibility strategy
- Set quality bar for Phase 4 and beyond

**Key Features Designed**:
- Real-time metrics display (FPS, throughput, IOPS, latency percentiles)
- Atomic operations for thread-safe counter updates
- Circular buffers for efficient memory management
- Sparkline visualization for trend analysis
- SSH-compatible terminal rendering
- Keyboard controls for interactive monitoring

**Released in**: v25.14.0 (Phase 4A - TUI Real-Time Dashboard)

**GitHub**: [@AnxietyLab](https://github.com/AnxietyLab)

## Recognition Levels

We recognize contributions at different levels:

- **Architecture & Design**: Significant architectural vision or design contributions
- **Core Maintainers**: Project leadership and long-term stewardship
- **Contributors**: Code, documentation, testing, or bug fixes
- **Community Members**: Issues, discussions, feedback, and support
